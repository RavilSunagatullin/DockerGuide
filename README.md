# DockerGuide

Мини-курс «Docker from zero to hero» для разработчиков. Каждый тематический блок хранится в отдельной директории на верхнем уровне репозитория и содержит:
- файл `theory.md` с подробной теорией;
- директорию `examples/` с разбором полезных команд и примеров;
- директорию `tasks/` с лабораторной работой и стартовым набором инструкций.

## Темы
- **getting-started/** — установка Docker и Docker Compose под Windows, macOS и Linux, проверка окружения и базовая ментальная модель контейнеров. Далее появятся блоки про жизненный цикл контейнера, образы, сеть и Compose.

## Как работать с курсом
1. Открывай тему по порядку, читай `theory.md`.
2. Разбирайся с примерами — повторяй команды у себя локально.
3. Выполняй задания из директории `tasks/`, сохраняй результаты в отдельной папке.
4. Переходи к следующему блоку только после того, как выполнишь лабораторное задание и ответишь на контрольные вопросы.

## Системные требования
- Docker Desktop (Windows/macOS) или Docker Engine (Linux) с включённой поддержкой Docker Compose.
- Доступ к интернету для загрузки образов из Docker Hub.
- Терминал с поддержкой Bash или PowerShell.
