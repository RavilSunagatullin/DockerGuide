# Примеры конфигураций

В этом каталоге собраны минимальные заготовки Dockerfile и сопутствующих файлов для разных языков. Они повторяют принципы из основной теории и предназначены для самостоятельных экспериментов.

- [`node-express/`](./node-express/) — Express-приложение на Node.js.
- [`go-http/`](./go-http/) — HTTP-сервер на Go.
- [`python-flask/`](./python-flask/) — Минимальный сервис на Flask.
- [`php/`](./php/) — Полная связка PHP + MariaDB и phpMyAdmin.

Каждая папка содержит Dockerfile, исходники приложения и файл `.dockerignore`.
